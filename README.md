## Features

### Real-Time Transcription
- **Dictation Mode** — Speak and see your words transcribed in real-time
- **Voice Typing** — Type anywhere on your Mac using your voice
- **Caption Mode** — Display live captions in a floating overlay window

### File Transcription
- **Batch Processing** — Transcribe multiple audio/video files at once
- **Translation** — Automatically translate transcripts using LLM providers
- **Timestamps** — Generate subtitles with accurate timing

### Multiple Speech Providers
- **On-Device** — Parakeet (Apple Silicon), WhisperKit, Apple Speech
- **Cloud** — OpenAI Whisper, Soniox, Deepgram, AssemblyAI, ElevenLabs

### Advanced Features
- **Speaker Diarization** — Identify different speakers in conversations
- **Custom Vocabulary** — Improve accuracy with domain-specific terms
- **Global Hotkeys** — Start/stop transcription from anywhere
- **Session History** — Review and export past transcriptions

## System Requirements

- macOS 14.0 (Sonoma) or later
- Apple Silicon recommended for on-device providers
- Intel Macs supported with cloud providers

## Installation

1. Download the latest release from the [Releases](https://github.com/tacshi/VoiceTale-Release/releases) page
2. Open the DMG file and drag VoiceTale to your Applications folder
3. Launch VoiceTale and grant the required permissions (Microphone, Screen Recording for system audio)

## Permissions

VoiceTale requires the following permissions:
- **Microphone** — For voice transcription
- **Screen Recording** — For capturing system audio (optional)
- **Accessibility** — For Voice Typing mode (optional)
